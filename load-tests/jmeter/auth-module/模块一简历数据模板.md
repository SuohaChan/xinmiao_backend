## 负例（未携带 Token）

未携带 Access Token 的并发请求均返回 **401**，与自定义 JWT 拦截器行为一致（JMeter 断言 HTTP 401，报告见 `auth-module-unauthorized`）。

## 续期（可选）

## 强制下线（功能为主，可无数字）
